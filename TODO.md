## 관전 흐름

- **`kbo recap <gameId>` — 종료 경기 정적 출력** — 🟢 · `관전` · added 2026-05-02
  - 끝난 경기의 텍스트 중계 + 박스스코어를 stdout 에 정적 출력.
  - `watch` 와 달리 alt-screen 을 쓰지 않아 파이프/스크롤 가능.
  - 데이터는 기존 `relay` 응답 (RESULT 상태) 재활용.
  - 효용: CLI 의 사용 시간대가 라이브 시간 밖으로 확장됨.

- **하이라이트 압축 모드 `watch --highlights`** — 🟡 · `관전` · added 2026-05-02
  - 득점 · 홈런 · 삼진 같은 이벤트만 누적 스크롤로 표시.
  - 회의 중 곁눈질용. watch 와 토글 가능 (`h` 키).
  - 이벤트 분류 로직이 핵심 (텍스트 매칭 vs 구조화 필드 확인 필요).

- **종료 직후 알림 `watch --notify`** — 🟢 · `관전` · added 2026-05-02
  - 경기 종료 시 OS 알림 한 번.
  - macOS: `osascript -e 'display notification ...'`, Linux: `notify-send` fallback.
  - 신규 런타임 의존성 없이 가능.

- **`kbo schedule --week`** — 🟢 · `관전` · added 2026-05-02
  - 7일치 일정을 표 한 장으로. 홈/원정 컬러칩 활용.
  - 휴식일/우천 표시 처리.

  - 게임로그가 별도 엔드포인트인지 사전 확인.

- **타구 분포 / 구종 비율 mini-chart** — 🟡 · `stats` · added 2026-05-02

## 개발자 편의

- **`kbo config` 서브커맨드 + 영구 설정** — 🟢 · `dx` · added 2026-05-02
  - `~/.config/kbo-cli/config.json` 에 즐겨찾는 팀 / 폴링 간격 / 테마 / 상태바 옵션 저장.
  - `kbo watch` 가 매번 `--team` 없이도 내 팀으로 시작.
  - 추후 첫 실행 시 인터랙티브 마법사로 확장 여지.

- **`kbo --tmux` 한 줄 statusline** — 🟢 · `dx` · added 2026-05-02
  - `LG 4-2 NC 7회초 ●●○` 형태로 stdout 한 번 출력 후 즉시 종료.
  - tmux `status-right` 등에서 주기적 호출용. 1초 미만 종료 보장.
  - 평소에 항상 보이게 되어 체감 가치가 큼.

- **fixture 골든 테스트** — 🟢 · `infra` · added 2026-05-02
  - `render:fixture` 출력 스냅샷을 `bun test` 에 묶어 회귀 방지.
  - 너비/색 옵션을 고정해 결정적 출력 보장.
  - CI 가치 큼.

- **`--json` 출력** — 🟢 · `dx` · added 2026-05-02
  - today / stats 에 `--json` 옵션. 정규화된 raw 결과를 stdout.
  - jq 등 외부 도구와 합성 가능.
  - TUI 경로와 분기만 잘 나누면 됨. 스키마는 README 에 명시.
